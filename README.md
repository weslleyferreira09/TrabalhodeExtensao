💎 Silv Store - Gestão de Estoque e Consultoria Digital
Este projeto foi desenvolvido como parte do Projeto de Extensão do curso de Engenharia de Software (Unopar). O objetivo é fornecer uma solução de controle de inventário e análise de vendas para a Silv Store, uma loja especializada em joias de prata.

🚀 Funcionalidades
Painel de Inventário: Visualização em tempo real do estoque de joias.

Alertas de Reposição: Sistema visual que indica quando itens estão com estoque baixo (Alerta Amarelo).

Cálculo de Faturamento: Processamento automático do valor total do estoque e lucros previstos.

Integração com Nuvem: Leitura dinâmica de dados diretamente do Google Sheets via CSV.

🛠️ Tecnologias Utilizadas
Linguagem: Python

Framework Web: Streamlit

Manipulação de Dados: Pandas

Versionamento: Git & GitHub

📊 Estrutura de Dados
O projeto utiliza conceitos fundamentais de Ciência da Computação, como:

Dicionários e Listas: Para mapeamento e organização dos produtos.

Busca Linear: Processamento das linhas da planilha para filtragem de itens.

Persistência de Dados: Uso de arquivos CSV (Valores Separados por Vírgulas).

🔧 Como Rodar o Projeto
Certifique-se de ter o Python instalado.

Instale as dependências:

Bash
pip install streamlit pandas
Execute a aplicação:

Bash
streamlit run silv.py
